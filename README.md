# Sistema de Gestión y Catálogo de Productos Personalizados

## Descripción

Plataforma web diseñada para emprendimientos que necesitan un **catálogo digital profesional**, con enfoque en conversión directa mediante WhatsApp.

Muchos emprendimientos no cuentan con una tienda online estructurada.
Este sistema centraliza productos en un catálogo organizado y facilita la venta directa.

## Funcionalidades

### Usuario

* Búsqueda en tiempo real
* Filtro por categorías
* Orden por precio
* Productos destacados
* Vista de producto
* Compra vía WhatsApp

### Administrador

* CRUD de productos
* Gestión de categorías
* Subida de imágenes
* Configuración de destacados

## Flujo de compra

```text
Usuario → Catálogo → Producto → WhatsApp → Compra
```

## Modelo de producto

```json
{
  "codigo": "PAP-001",
  "nombre": "Agenda personalizada",
  "precio": 8500,
  "categoria": "Agendas",
  "destacado": true
}
```
